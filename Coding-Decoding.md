# 🔑 Coding-Decoding — Shortcuts

## Types of Coding

### 1. Letter-Shift Coding
Each letter shifted by a fixed number.

> "CAT" is coded as "DBU" → each letter +1
> A→B, C→D, T→U ✅

**Trick:** Write A=1, B=2 ... Z=26. Find the shift pattern.

### 2. Reverse Alphabet
A↔Z, B↔Y, C↔X ... (opposite letters)

> Formula: Opposite of letter N = 27 - N
> A(1) ↔ Z(26) → 1+26=27 ✓

### 3. Position-Based Coding
Letters coded by their position numbers.

> "CAT" → 3, 1, 20

### 4. Mixed/Random Coding
Look for: same letter in coded word, consistent substitution.

> If "FORM" = "GPSN" → each letter +1
> Verify: F+1=G ✓, O+1=P ✓, R+1=S ✓, M+1=N ✓

## Step-by-Step Method

1. Write out letter positions (A=1 to Z=26)
2. Find difference between original and coded letter
3. Check if difference is consistent
4. Apply to unknown word
