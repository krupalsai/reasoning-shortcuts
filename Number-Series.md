# 🔢 Number Series — Shortcuts

## Common Patterns

### 1. Arithmetic Series (same difference)
> 2, 5, 8, 11, 14 ... → difference = +3

### 2. Geometric Series (same ratio)
> 2, 6, 18, 54 ... → ratio = ×3

### 3. Square Series
> 1, 4, 9, 16, 25 ... → n²
> 2, 5, 10, 17, 26 ... → n² + 1

### 4. Cube Series
> 1, 8, 27, 64, 125 ... → n³

### 5. Fibonacci-like
> 1, 1, 2, 3, 5, 8, 13 ... → each = sum of previous two

### 6. Difference of Differences
> 1, 2, 4, 7, 11, 16 ...
> Differences: 1, 2, 3, 4, 5 → increasing by 1

### 7. Alternating Series
> 2, 10, 4, 8, 6, 6, 8, ? → two interleaved series!
> Series 1: 2, 4, 6, 8 (+2)
> Series 2: 10, 8, 6, **4** ✅

## Quick Strategy

1. Find differences between consecutive terms
2. If not constant → find differences of differences
3. Check if squares, cubes, or primes
4. Look for alternating patterns

## Practice

1. 3, 6, 12, 24, ___ → *48 (×2)*
2. 1, 4, 9, 16, 25, ___ → *36 (n²)*
3. 2, 3, 5, 7, 11, ___ → *13 (primes)*
4. 1, 2, 4, 7, 11, ___ → *16 (diff +1,+2,+3,+4,+5)*
