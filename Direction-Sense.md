# 🧭 Direction Sense — Shortcuts

## Compass Diagram (always draw this!)

```
           NORTH
             ↑
    WEST ←──┼──→ EAST
             ↓
           SOUTH
```

Diagonals: NE, NW, SE, SW

## Turn Rules

| Action | Result |
|--------|--------|
| Face North, turn Right | Face East |
| Face North, turn Left | Face West |
| Face East, turn Right | Face South |
| Turn 180° | Opposite direction |

**Clockwise turns:** N → E → S → W → N

## Distance (Pythagoras Trick)

If you go **a** meters one direction and **b** meters perpendicular:
```
Distance from start = √(a² + b²)
```

> Walk 3m East, then 4m North → Distance = √(9+16) = √25 = **5m**

## Shadow Trick

| Time of day | Sun is in | Shadow falls |
|-------------|-----------|--------------|
| Morning | East | West |
| Evening | West | East |

## Step-by-Step Method

1. Draw a coordinate grid
2. Mark starting point as origin
3. Plot each movement carefully
4. Calculate final distance/direction

## Practice

1. Start facing North. Turn right 90°, walk 5m. Turn right 90°, walk 12m. Distance from start?
   → *13m (5-12-13 triangle)*

2. A walks 10m East, then 10m South, then 10m West. Where is he relative to start?
   → *10m South*
